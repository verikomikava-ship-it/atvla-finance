# Atvla Finance - პროექტის სრული სპეციფიკაცია დიზაინერისთვის

## პროექტის შესახებ
**სახელი:** Atvla Finance (ჩემი ფინანსები)
**ტექნოლოგია:** React + TypeScript + Tailwind CSS + Firebase
**პლატფორმა:** Web (Vercel), Desktop (Electron), Mobile-responsive
**ენა:** ქართული (Georgian)
**თემა:** Dark / Light mode

## 1. HEADER კომპონენტი

### ელემენტები:
- **თვის სელექტი** — dropdown, 12 თვე + "ყველა თვე"
- **შემოსავალი** — მწვანე ფერი, ₾ ნიშანი
- **ხარჯი** — წითელი ფერი
- **ბალანსი** — (შემოსავალი - ხარჯი), მწვანე/წითელი
- **მიზნის ტრეკერი** — პროგრესბარი + სახელი + თანხა
- **BillAlerts** — გადახდის გაფრთხილებები (წითელი badge-ები)

### ფერები:
- შემოსავალი: `emerald-600`
- ხარჯი: `red-500`
- ბალანსი: მწვანე თუ > 0, წითელი თუ < 0
- ფონი: გრადიენტი `blue-600 → emerald-600`

## 2. SmartAdvisor (ჭკვიანი მრჩეველი)

### Default: **Collapsed** (ჩაკეცილი)
```
┌─────────────────────────────────────────────────┐
│ 🧠 ჭკვიანი მრჩეველი          [▼ გახსნა]        │
│                                                  │
│  [85 ❤️]  💰 450₾ ჯიბის  📊 45₾/დღე            │
│           [🔴 2] [🟡 3] [🟢 5]                   │
│                                                  │
│  ⚠️ კრიტიკული: ქირის ვადა 2 დღეში!             │
│  ⚠️ კრიტიკული: სესხის გადახდა ხვალ!             │
└─────────────────────────────────────────────────┘
```

### Expanded (გახსნილი):
- **ჯანმრთელობის ქულა** — წრიული ინდიკატორი (0-100)
  - 80+ მწვანე, 50-79 ყვითელი, <50 წითელი
- **50/30/20 ბარი** — საჭირო/სურვილი/დანაზოგი
- **შემოსავალი vs ვალდებულებები** — შედარების ბარი
- **4 მინი მეტრიკა** — ხარჯის რეიტი, დანაზოგი, ვალის რეიტი, თავისუფალი
- **ინსაიტები** — ტექსტური რჩევები (max 4, "მეტი" ღილაკი)
- **სესხების ინფო** — თვეები დარჩენილი, გადახდილი
